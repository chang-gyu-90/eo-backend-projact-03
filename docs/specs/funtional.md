## 야구 커뮤니티 단일 게시판 사이트
기능 명세서 작성

## 1. 회원가입
- 설명 : 새로운 회원을 등록한다.
- 요청 URL : /account/signup
- 요청 방식 : POST
- 요청 매개 변수 : 아이디(user_id), 비밀번호(password), 이메일(email), 닉네임(nickname)
- 입력 양식 :
- 기능 수행 후 URL 재지정 : /account/login
- 제약 조건
    - 아이디, 비밀번호, 이메일, 닉네임은 필수 입력해야 한다.
        - 오류 메시지 : 필수 입력 항목은 반드시 입력해야 합니다.
    - 아이디는 50자 이하여야 한다.
        - 오류 메시지 : 아이디는 50자 이하여야 합니다.
    - 아이디는 중복될 수 없다.
        - 오류 메시지 : 이미 사용중인 아이디입니다.
    - 비밀번호는 50자 이하여야 한다.
        - 오류 메시지 : 비밀번호는 50자 이하여야 합니다.
    - 이메일은 올바른 형태로 입력해야 한다.
        - 오류 메시지 : 올바른 이메일을 입력해야 합니다.
    - 이메일은 20자 이하여야 한다.
        - 오류 메시지 : 이메일은 20자 이하여야 합니다.
    - 이메일은 중복될 수 없다.
        - 오류 메시지 : 이미 사용중인 이메일입니다.
    - 닉네임은 50자 이하여야 한다.
        - 오류 메시지 : 닉네임은 50자 이하여야 합니다.

## 2. 로그인
- 설명 : 아이디와 비밀번호로 로그인하고 JWT 토큰을 발급받는다.
- 요청 URL : /account/login
- 요청 방식 : POST
- 요청 매개 변수 : 아이디(user_id), 비밀번호(password)
- 입력 양식 :
- 기능 수행 후 URL 재지정 : /board/list
- 제약 조건
    - 아이디, 비밀번호는 필수 입력해야 한다.
        - 오류 메시지 : 아이디, 비밀번호를 반드시 입력해야 합니다.
- 기능
    - 아이디, 비밀번호 일치 시 로그인 성공, JWT 토큰 발급
    - 아이디, 비밀번호 불일치 시 오류 출력
        - 오류 메시지 : 아이디 또는 비밀번호가 올바르지 않습니다.

## 13. 게시물 작성
- 설명 : 새로운 게시물을 등록한다.
- 요청 URL : /board/write
- 요청 방식 : POST
- 요청 매개 변수 : 게시물 제목(title), 게시물 내용(content), 카테고리(category)
- 입력 양식 : 필요(GET /board/write)
- 기능 수행 후 URL 재지정 : /board/read
- 제약 조건
    - 비로그인 사용자는 게시물 작성에 접근할 수 없다.
        - 오류 메시지 : 로그인이 필요합니다.
    - 제목은 필수 입력해야 한다.
        - 오류 메시지 : 제목과 내용은 반드시 입력되어야 합니다.
    - 제목은 100자 이하여야 한다.
        - 오류 메시지 : 제목은 100자 이하여야 합니다.
    - 내용은 필수 입력해야 한다.
        - 오류 메시지 : 제목과 내용은 반드시 입력되어야 합니다.
- 기능
    - 사용자는 게시물을 등록할 수 있다.
    - 게시물에 첨부 파일을 추가할 수 있다.
        - 첨부 파일의 용량은 제한 이내여야 한다.
        - 오류 메시지 : 용량이 초과되었습니다.

        - 오류 메시지 : 로그인 후 사용하여 주시기 바랍니다.
    - 등록된 id여야 한다.
        - 오류 메시지 : 존재하지 않거나 삭제된 게시물입니다.
    - 동일 사용자가 같은 게시물에 중복 등록할 수 없다.
        - 오류 메시지 : 이미 좋아요를 누른 게시물입니다.
- 기능
    - 좋아요 등록 시 해당 게시물의 likes 값이 1 증가한다.
