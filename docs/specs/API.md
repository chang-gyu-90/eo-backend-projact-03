## 야구 커뮤니티 단일 게시판 사이트
게시판 및 게시물, 댓글, 좋아요 기능을 제공한다.

사용자는 게시물을 생성, 조회, 수정, 삭제할 수 있으며, 게시판 단위로 게시물을 관리한다.
사용자는 게시물에 댓글을 등록, 수정, 삭제할 수 있다.
사용자는 게시물에 좋아요를 등록하거나 취소할 수 있다.

### 2. 공통 요청 / 응답 형식
    - application/json

### 3. 인증 / 인가 방식
    - 게시물 목록 조회, 게시물 상세 조회, 댓글 목록 조회 API는 로그인 없이 가능하다.
    - 게시물 작성, 수정, 삭제 API는 로그인이 필수이다.
    - 댓글 작성, 수정, 삭제 API는 로그인이 필수이다.
    - 좋아요 등록, 취소 API는 로그인이 필수이다.
    - JWT(Authorization Header)

### 4. API 사용 순서
1. 게시판 목록 조회 (인증 불필요)
2. 게시물 목록 조회 (인증 불필요)
3. 게시물 상세 조회 (인증 불필요)
4. 댓글 목록 조회 (인증 불필요)
5. [인증 필요] 회원가입 / 로그인 (JWT 토큰 발급)
6. [인증 필요] 게시물 작성
7. [인증/인가 확인] 게시물 수정 (본인 작성 게시물만)
8. [인증/인가 확인] 게시물 삭제 (본인 작성 게시물만)
9. [인증 필요] 댓글 작성
10. [인증/인가 확인] 댓글 수정 (본인 작성 댓글만)
11. [인증/인가 확인] 댓글 삭제 (본인 작성 댓글만)
12. [인증 필요] 좋아요 등록 / 취소

### 5. API Reference
    1. 회원 관리 (Account)
    2. 게시판 관리 (Boards)
    3. 게시물 관리 (Posts)
    4. 댓글 관리 (Comments)
    5. 좋아요 관리 (Likes)
 


#### 회원 관리

| 기능 설명       | 요청 방식 | URL               | JSON 포함 내용                          | 제약 조건 및 응답                                   |
|:-----------:|:-----:|-------------------|-------------------------------------|------------------------------------------------|
| 회원가입        | POST  | /account/signup   | user_id, password, email, nickname  | user_id·email 중복 불가, 각 필드 글자수 제한              |
| 로그인         | POST  | /account/login    | user_id, password                   | 일치 시 JWT 발급, 불일치 시 401                        |
| 로그아웃        | POST  | /account/logout   | refresh_token                       | 로그인 상태여야 함, Refresh Token 무효화                  |
| 사용자 정보 조회   | GET   | /account/info     |                                     | 로그인 상태여야 함, 비밀번호 필드 제외                        |
| 사용자 정보 수정   | POST  | /account/update   | password, nickname                  | 로그인 상태여야 함, 닉네임 중복 불가, 닉네임 50자 이하             |
| 회원 탈퇴       | POST  | /account/leave    | password                            | 비밀번호 재입력 필수                                    |
 


#### 게시판 관리

| 기능 설명       | 요청 방식 | URL                              | JSON 포함 내용 | 제약 조건 및 응답                              |
|:-----------:|:-----:|----------------------------------|------------|-------------------------------------------|
| 게시판 목록 조회   | GET   | /boards                          |            | 비회원 조회 가능                                |
| 게시판 생성      | POST  | /admin/board/create              | name       | 관리자만 가능, 이름 20자 이하, 필수 입력                 |
| 게시판 수정      | POST  | /admin/board/{board_id}/update   | name       | 관리자만 가능, 등록된 board_id여야 함                 |
| 게시판 삭제      | POST  | /admin/board/{board_id}/delete   |            | 관리자만 가능, 삭제 시 하위 게시물·댓글 함께 삭제             |
 


#### 게시물 관리

| 기능 설명       | 요청 방식 | URL            | JSON 포함 내용              | 제약 조건 및 응답                                         |
|:-----------:|:-----:|----------------|-------------------------|------------------------------------------------------|
| 게시물 목록 조회   | GET   | /board/list    |                         | 페이지(page), 게시판(board_id) 파라미터, 제목 옆 댓글 수 표시         |
| 게시물 상세 조회   | GET   | /board/read    |                         | 등록된 id여야 함 (존재하지 않을 시 오류), 조회 시 views +1             |
| 게시물 작성      | POST  | /board/write   | title, content, category | 제목·내용 필수 입력, 제목 100자 이하, 비로그인 작성 불가, 첨부파일 추가 가능     |
| 게시물 수정      | POST  | /board/update  | id, title, content, category | 제목·내용 필수 입력, 제목 100자 이하, 작성자 본인만 수정 가능          |
| 게시물 삭제      | POST  | /board/delete  | id                      | 등록된 id여야 함 (존재하지 않을 시 오류), 작성자 본인만 삭제 가능, 댓글·좋아요 함께 삭제 |
 


#### 댓글 관리

| 기능 설명       | 요청 방식 | URL                                   | JSON 포함 내용              | 제약 조건 및 응답                                    |
|:-----------:|:-----:|---------------------------------------|-------------------------|--------------------------------------------------|
| 댓글 목록 조회    | GET   | /board/{id}/comment/list              |                         | 비회원 조회 가능, 등록된 id여야 함                           |
| 댓글 작성       | POST  | /board/{id}/comment/write             | content                 | 내용 1~1000자, 비로그인 작성 불가                          |
| 댓글 수정       | POST  | /board/{id}/comment/update            | comment_id, content     | 내용 1~1000자, 작성자 본인만 수정 가능                       |
| 댓글 삭제       | POST  | /board/{id}/comment/delete            | comment_id              | 등록된 comment_id여야 함 (존재하지 않을 시 오류), 작성자 본인만 삭제 가능 |
 


#### 좋아요 관리

| 기능 설명       | 요청 방식 | URL                       | JSON 포함 내용 | 제약 조건 및 응답                                        |
|:-----------:|:-----:|---------------------------|------------|----------------------------------------------------|
| 좋아요 등록      | POST  | /board/{id}/like          |            | 로그인 필수, 등록된 id여야 함, 중복 등록 불가, posts.likes +1       |
| 좋아요 취소      | POST  | /board/{id}/like/cancel   |
